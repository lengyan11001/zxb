## 回滚思路

查看提交：

```bash
git log --oneline -10
```

回到指定版本：

```bash
git reset --hard <commit>
npm ci
npm run build
systemctl restart zxb-intel.service zxb-intel-worker.service
```

只在明确需要回滚时使用 `git reset --hard`。
